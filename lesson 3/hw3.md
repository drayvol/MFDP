## 1. Целевая аудитория и проблема

Целевая аудитория продукта:

- ученики 10–11 классов, готовящиеся к ЕГЭ по профильной математике;
- репетиторы по математике;
- школьные учителя.

Проблема:

При самостоятельной подготовке ученик часто не может быстро определить, правильно ли решено задание и на каком шаге была допущена ошибка. Проверка преподавателем требует времени и денежных затрат, а существующие математические сервисы обычно показывают готовое решение, но не анализируют решение самого ученика.

Будущий продукт:

Telegram-бот, который принимает фотографию решения задания №13 ЕГЭ, распознает математические выражения, проверяет ход решения, находит ошибку в конкретном шаге, выставляет оценку 0/1 и рекомендует похожие задачи для отработки выявленной ошибки.

---

## 2. Типы конкурентов и бенчмарков

Для анализа выбраны три типа аналогов.

### 2.1. Прямой аналог

Репетитор или эксперт ЕГЭ.

Репетитор решает ту же задачу для той же аудитории: проверяет решение ученика, выставляет балл и объясняет ошибки.

Сильная сторона — высокая точность проверки и персональная обратная связь.

Слабая сторона — высокая стоимость, ограниченная доступность и невозможность масштабирования.

---

### 2.2. Косвенные аналоги

Photomath и Mathway.

Photomath используется как пример качественного OCR математических выражений и удобной работы с фотографиями задач.

Mathway используется как пример автоматического решения математических задач различных типов.

Оба продукта позволяют быстро получить решение задачи, однако не проверяют решение ученика и не оценивают его по критериям ЕГЭ.

---

### 2.3. Отраслевой AI-бенчмарк

Яндекс Учебник — Репетитор AI.

Сервис используется как пример современного AI-помощника в образовании, который способен объяснять решение задач, вести диалог с учеником и формировать персонализированный план подготовки.

Для нашего проекта этот продукт является ориентиром по качеству объяснений и пользовательскому опыту.

## 4. Значимые параметры продукта

На основе анализа конкурентов важны следующие параметры:

- F1-score итоговой оценки решения;
- качество OCR-распознавания рукописных математических выражений (CER);
- успешность обработки фотографий;
- качество рекомендаций похожих задач;
- понятность объяснения ошибки;
- охват типов уравнений задания №13;
- удобство использования через Telegram.

---

## 5. Значения бенчмарков и источники

### 5.1. Репетитор / эксперт ЕГЭ

Ручная проверка обеспечивает практически максимальную точность оценки решения и качественную обратную связь.

Найденные значения:

- точность проверки близка к 100%;
- стоимость 1500–3000 рублей за час;
- время ожидания проверки от нескольких часов до нескольких дней.

Для нашего проекта это benchmark по качеству проверки и объяснению ошибок.

---

### 5.2. Photomath

Photomath является одним из наиболее популярных сервисов распознавания и решения математических задач по фотографии.

Найденные характеристики:

- работа с фотографиями рукописных выражений;
- широкий охват математических тем;
- получение результата за несколько секунд.

Для нашего проекта это benchmark по удобству загрузки фотографий и качеству OCR.

---

### 5.3. Mathway

Mathway представляет собой один из наиболее распространенных автоматических математических решателей.

Найденные характеристики:

- поддержка алгебры, тригонометрии и математического анализа;
- автоматическое построение решения;
- получение ответа за несколько секунд.

Для нашего проекта это benchmark по автоматической обработке математических выражений.

---

### 5.4. Яндекс Учебник — Репетитор AI

Сервис использует искусственный интеллект для объяснения решений и сопровождения подготовки к ЕГЭ.

Найденные характеристики:

- охват всего банка заданий ФИПИ;
- диалоговый формат взаимодействия;
- персонализированные рекомендации.

Для нашего проекта это benchmark по качеству объяснений и образовательной ценности продукта.

## 6. Минимальные требования к нашему продукту

### Функциональные требования

- загрузка фотографии решения;
- OCR математических выражений;
- проверка решения задания №13;
- локализация ошибки в конкретном шаге;
- бинарная оценка решения (0/1);
- подбор похожих задач.

### ML-бенчмарки

- F1-score > 0.75;
- CER OCR < 15%;
- успешная обработка фотографий ≥ 95%;
- Recall@3 > 0.7 для рекомендательной системы.

### Продуктовые бенчмарки

- время ответа менее 5 минут;
- покрытие основных типов уравнений задания №13;
- понятность объяснения ошибки не ниже 4/5 по результатам пользовательского тестирования;
- возможность проверки решения без участия преподавателя.

## 8. Итог

Главный стандарт продукта заключается не в автоматическом решении математической задачи, а в автоматической проверке решения ученика.

Минимальный целевой формат продукта:

Telegram-бот, который по фотографии решения задания №13 ЕГЭ автоматически проверяет ход решения, локализует ошибку, выставляет оценку и рекомендует похожие задачи для дальнейшей практики.

Ключевое конкурентное преимущество продукта:

проверка решения ученика по шагам с объяснением ошибок — функциональность, отсутствующая у рассмотренных конкурентов.
